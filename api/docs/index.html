Chat Room API Reference

POST /api/rooms/
Creates a new chat room.

Request Body

task required
Object containing task details:
{
  "title": "string",
  "description": "string",
  "objectives": ["string"],
  "deadline": "string (ISO 8601)"
}
personality required
Object containing AI personality settings:
{
  "name": "string",
  "description": "string",
  "traits": {
    "emotional_stability": {"level": 0.8, "description": "string"},
    "extraversion": {"level": 0.7, "description": "string"},
    ...
  },
  "communication_style": {
    "formality": 0.7,
    "directness": 0.8,
    ...
  }
}
participants required
Array of participant objects:
[{
  "user_id": "string",
  "name": "string",
  "role": "string",
  "is_ai": boolean
}]

Response

{
  "room_id": "string",
  "task": object,
  "personality": object,
  "participants": array,
  "created_at": "string (ISO 8601)",
  "status": "string",
  "join_url": "string"
}
GET /api/rooms/{room_id}
Retrieves details of a specific chat room.

Path Parameters

room_id required
The unique identifier of the chat room.

Response

{
  "room_id": "string",
  "task": object,
  "personality": object,
  "participants": array,
  "created_at": "string",
  "status": "string",
  "join_url": "string"
}
GET /api/rooms/list
Lists all chat rooms or rooms for a specific user.

Query Parameters

user_id optional
Filter rooms by user ID. If not provided, returns all rooms.

Response

[{
  "room_id": "string",
  "task": object,
  "personality": object,
  "participants": array,
  "created_at": "string",
  "status": "string",
  "join_url": "string"
}]
POST /api/rooms/{room_id}/messages
Sends a message to a chat room.

Path Parameters

room_id required
The unique identifier of the chat room.

Request Body

{
  "user_id": "string",
  "content": "string",
  "type": "string",
  "role": "string" // optional
}

Response

{
  "user_id": "string",
  "channel_name": "string",
  "content": "string",
  "ts": number,
  "role": "string",
  "type": "string"
}
GET /api/rooms/{room_id}/messages
Retrieves all messages from a chat room.

Path Parameters

room_id required
The unique identifier of the chat room.

Response

[{
  "user_id": "string",
  "channel_name": "string",
  "content": "string",
  "ts": number,
  "role": "string",
  "type": "string"
}]
GET /chat/{room_id}
Joins a chat room via web interface.

Path Parameters

room_id required
The unique identifier of the chat room.

Response

Returns HTML page for chat interface
